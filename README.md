# CRGkit
Toolkit to work with Curved Regular Grids based on the OpenCRG library.

## Tools

### crg2obj
Convert OpenCRG model to Wavefront OBJ

Usage:
```
cgr2obj [options] <filename>
 options: -h            show this info
 options: -v offset     apply transverse offset
 <filename>  input file
```
